## 시작하기

아래의 내용을 따라 프로젝트를 로컬 환경에서 실행할 수 있습니다.

### 환경

- Java 17 
- spring boot 3.1.2
- gradle 8.2.1

### 설치 및 실행

1. 저장소 클론 

```bash
git clone https://github.com/illhab/ilhab_backend.git
```

2. intellij java version 17로 설정 

3. gradle refresh
